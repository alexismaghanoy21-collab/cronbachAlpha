Formula & Definition
α = (k / (k − 1)) × (1 − Σσ²ᵢ / σ²ₜ)
k — Number of items in the scale
σ²ᵢ — Population variance of each item
Σσ²ᵢ — Sum of all item variances
σ²ₜ — Variance of the total composite scores
α — Cronbach's Alpha (0 to 1)
Reliability Interpretation
α ≥ 0.90 Excellent
0.80 ≤ α < 0.90 Good
0.70 ≤ α < 0.80 Acceptable
0.60 ≤ α < 0.70 Questionable
0.50 ≤ α < 0.60 Poor
α < 0.50 Unacceptable
